FHX Parsing Tool (Bachelorproef)

Voor mijn bachelorproef moest ik een project op een stage uitvoeren en hierover een bachelorproef schrijven. De probleemstelling van het bedrijf was als volgt: "Een programma in DeltaV is zeer complex en vrij groot. Aanpassingen hieraan vragen veel tijd en zijn zeer arbeidsintensief. Er is nood aan overzicht en automatisatie hiervoor." DeltaV is een programma voor, kort gezegd, het automatiseren van een fabriek. Aan de FHX Parsing Tool moest alle data hieruit worden weggeschreven naar een database.

Voor meer informatie over de bachelorproef heb ik hieronder enkele nuttige links. De code zelf kan ik echter niet vrijgeven maar mijn bachelorproef zelf wel. Alsook de powerpoint van mijn eindverdediging en de poster van het project.
