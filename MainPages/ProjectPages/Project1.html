Portfoliosite

Voor het vak "User Interfaces" moest ik een website maken. Als u dit leest is dat teken dat hij al zeker en vast goed te bereiken is. Het maken van deze website zorgde ervoor dat ik vertrouwd geraakte met de basis van html en css. Ook komt er enige php aan te pas maar dit was echter zeer beperkt. Eerst moesten we een wireframe maken van de website. Dit betekent dat we een voorbeeld moesten maken van hoe onze site eruit moest zien. Dit moest niet gebeuren met html en css maar met een visueel programma. Zelf heb ik dit gedaan met "Lunacy". Dit werkte zeer gemakkelijk en zorgde ervoor dat ik een goed beeld had van wat ik wou gaan maken. Hieronder een voorbeeld van de wireframe van mijn startpagina.

Daarna kwamen er nog vele zaken bij. Zo moest de site onder andere een projectpagina bevatten waar u zich nu op bevindt. Voor de code van de website zelf kan u hieronder de github link terugvinden.
